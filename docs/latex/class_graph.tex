\doxysubsection{Detailed Description}
Class representing a \mbox{\hyperlink{class_graph}{Graph}}. 

This class represents a \mbox{\hyperlink{class_graph}{Graph}} with a vector of its nodes and a vector of vectors of its edges, as well as the name of the files from which the nodes and edges are loaded.

Uses the Ant Colony Optimization (ACO) on a distance matrix to find the best path. 

This functions uses the Ant Colony Optimization algorithm to find the best path in a given distance matrix. It does this by initializing a pheromone matrix, creating ant paths, updating the pheromone levels and then iterates for a given amount of maximum iterations. Once finished, returns the best path found.

Calculates the value of a given path in the graph. 

This function calculates the value of a given path represented by a vector of integers in the graph. The value is calculated by adding up the distances between consecutive nodes in the path using the \mbox{\hyperlink{class_graph_ac2702571c28c29a5157ec34590439d1d}{Graph\+::get\+Distance}} function. Optionally, it can also print the intermediate sums if the {\ttfamily out} parameter is set to {\ttfamily true}.
