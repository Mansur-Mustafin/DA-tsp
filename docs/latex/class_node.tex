\doxysubsection{Detailed Description}
Class representing a node of a \mbox{\hyperlink{class_graph}{Graph}}. 

This class represents the node of a \mbox{\hyperlink{class_graph}{Graph}} and has attributes, id, latitude and longitude. This class represents real global positions, represented by the attributes latitude and longitude. The attribute id allows the node to be easily identified.
