\doxysubsection{Detailed Description}
Class representing a \mbox{\hyperlink{class_graph}{Graph}}. 

This class represents a \mbox{\hyperlink{class_graph}{Graph}} with a vector of its nodes and a vector of vectors of its edges, as well as the name of the files from which the nodes and edges are loaded.
